NewsDigest
Target users: Busy professionals, info addicts who know they're wasting time
Core loop:

Ingest RSS feeds, news APIs, or user-pasted articles
Strip: repetition, emotional activation language, undefined sources, speculation
Output: Only falsifiable claims, novel information, named sources
Daily briefing: "Here's what actually happened in 847 words"

Monetization:

Free tier: 3 sources, daily digest
Pro tier: $10/month, unlimited sources, real-time, custom topics
API: For newsrooms and researchers who want the filter

Killer feature: Side-by-side mode. Show original article next to compressed version. Let users see the 94% filler. Once they see it, they're hooked.
